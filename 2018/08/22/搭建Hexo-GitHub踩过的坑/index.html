一、Hexo是什么？

Hexo官网点这里

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

其实说白了就是将我们所熟悉的写博客文件的格式.md解析成了html，服务数据什么的官方已经帮助解决，我们再将其托管在GitHub Pages上，这样就可以实现完全免费构建博客，当然推荐爬梯，否则会很慢
